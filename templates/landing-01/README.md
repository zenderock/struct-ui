# Struct UI - Starter Template

This is the primary starter template for Struct UI. It provides a "Linear-like" dashboard layout with a highly responsive sidebar, data tables, and high-density UI patterns.

![Dashboard Preview](./dashboard-preview.png)

## Features

- **Responsive Sidebar**: Collapsible, accessible, and motion-driven navigation.
- **Data Tables**: High-performance tables using TanStack Table & Virtualization.
- **Search Params**: Deep integration with URL search params for filtering/sorting.
- **Motion System**: Polished interactions using specific bezier curves (`ease-struct`).

## Tech Stack

- TanStack Start
- TanStack Router
- Tailwind CSS
- Base UI

## Getting Started

1. Install dependencies:
   ```bash
   pnpm install
   ```

2. Run development server:
   ```bash
   pnpm dev
   ```
